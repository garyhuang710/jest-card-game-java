Jest

Cette application implemente le jeu de cartes Jest. Elle comprend un moteur de jeu, des regles et variantes, un calcul de score, ainsi que deux interfaces utilisateur (CLI et GUI Swing).

Composants principaux

  • Modele de domaine : cartes, joueurs, offres, jest.
  • Moteur de jeu : distribution, prises, attribution de trophees.
  • Regles et variantes : comparaison, ordre de jeu, scores.
  • Interfaces utilisateur : CLI et GUI.
